# Knowledge Base

> 本目录由 ewankb 自动构建和管理，包含从源码和文档提炼的结构化业务知识。

## 目录结构

```
knowledgeBase/
  _meta/          元数据（索引、状态）
  需求文档/       业务需求相关文档
  接口文档/       API 和接口说明
  业务规则/       业务校验规则和规范
  测试用例/       验收测试用例
  研发设计文档/   技术设计和架构文档
  其他/           未分类文档
```

## 使用方式

运行 `ewankb build` 将自动从 `source/` 和 `domains/` 提炼文档到本目录。

运行 `ewankb query-kb "你的问题"` 可以检索本目录中的文档。
